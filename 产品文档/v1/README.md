## V1 版本概述

### 核心功能

- ✅ 用户认证（登录/注册）
- ✅ NPC 管理（创建、列表、详情）
- ✅ AI 对话（用户与单一 Agent 对话）
- ✅ 对话历史（历史记录、会话列表）
- ✅ Markdown 渲染（代码高亮、表格等）
- ✅ 移动端自适应
- ✅ Docker 部署

### 目标用户

**主要用户群体**：我的朋友

V1 版本主要面向朋友群体，用于验证 AI NPC 一对一对话的核心价值。

### 产品定位

**核心场景**：用户与单一 Agent 对话

V1 版本专注于用户与单个 AI NPC 的一对一对话体验，不涉及多 NPC 交互、NPC 互相对话等高级功能。

### 产品核心承诺

1. **数据永久保留**：永久保留用户数据，除非用户主动删除
2. **Agent 持续优化**：随着公司和技术发展，持续优化已有 Agent

---

## 技术栈

- **前端**：React 18 + Vite 5 + Ant Design
- **后端**：Node.js + Express.js
- **数据库**：MySQL 8.0
- **部署**：Docker + Docker Compose + Nginx
