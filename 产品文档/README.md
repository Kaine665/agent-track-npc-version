## 📋 文档说明

### V1 版本状态

- **版本**：v1.0.0
- **状态**：✅ 已完成并部署
- **完成时间**：2025-11-22
- **目标用户**：我的朋友
- **核心场景**：用户与单一 Agent 对话

### V1 核心功能

- ✅ 用户认证（登录/注册）
- ✅ NPC 管理（创建、列表、详情）
- ✅ AI 对话（用户与单一 Agent 对话）
- ✅ 对话历史（历史记录、会话列表）
- ✅ Markdown 渲染（代码高亮、表格等）
- ✅ 移动端自适应
- ✅ Docker 部署

### V1 产品核心承诺

1. **数据永久保留**：永久保留用户数据，除非用户主动删除
2. **Agent 持续优化**：随着公司和技术发展，持续优化已有 Agent
