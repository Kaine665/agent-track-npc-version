## v1.0.0 (2025-11-22)

### 核心功能
- ✅ 用户认证（登录/注册）
- ✅ NPC 管理（创建、列表、详情）
- ✅ AI 对话（支持多 LLM 提供商）
- ✅ 对话历史（历史记录、会话列表）
- ✅ Markdown 渲染（代码高亮、表格等）
- ✅ 移动端自适应
- ✅ Docker 部署（生产环境）

### 技术栈
- 前端：React 18 + Vite 5 + Ant Design
- 后端：Node.js + Express.js + MySQL
- 部署：Docker + Docker Compose + Nginx
