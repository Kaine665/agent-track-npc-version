## ❓ 遇到问题？

1. **查看日志**：`docker-compose logs -f`
2. **查看服务状态**：`docker-compose ps`
3. **查看详细文档**：对应的部署文档中有故障排查部分
4. **重启服务**：`docker-compose restart`
